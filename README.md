# Information
O objetivo é aprender a conectar em outra máquina local.  

**Rede**: Local  
**Sistema operacional**: Ubuntu  

**Client**: Máquina que vai acessar outra  
**Server**: Máquina que vai ser acessada  

**Descobrir o IP Local**: `ip addr show`
